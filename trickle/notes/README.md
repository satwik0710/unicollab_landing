# UniCollab

A professional, high-performance landing page for a university collaboration platform.

## Overview
UniCollab is designed to bridge the gap between academic potential and real-world project execution. The platform facilitates student collaboration across departments.

## Tech Stack
- React 18
- Tailwind CSS (Dark Mode optimized)
- Lucide Icons

## Key Features
- **Dark Mode / Cosmic Theme**: Unified immersive background across all pages.
- **Glassmorphism UI**: Modern, sleek interface tailored for dark backgrounds.
- **Core Modules**: Bento grid showcasing platform capabilities.
- **Interactive Chatbot**: Simulated AI mentor interface.
- **Project Marketplace**: Carousel of student projects.
